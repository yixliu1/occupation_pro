<h1>Occupation Name: Web Developer</h1>
<h1>ANZSCO Code: 261212</h1>
<h1>Assessment Authorities: ACS</h1>
<h2>Job description </h2>
<p>Web Developer Plans, produces and maintains web sites using web programming languages, software applications, technologies and databases together with specifications of user needs, often in conjunction with other ICT Professionals such as Business Analysts, Web Designers and network and usability specialists.</p>
<p>Web Developer may perform the following tasks:</p>
<li>analysing, designing and developing Internet sites applying a mixture of artistry and creativity with software programming and scripting languages and interfacing with operating environments</li>
<li>designing and developing digital animations, imaging, presentations, games, audio and video clips, and Internet applications using multimedia software, tools and utilities, interactive graphics and programming languages</li>
<li>communicating with network specialists regarding web-related issues, such as security and hosting web sites, to control and enforce Internet and web server security, space allocation, user access, business continuity, web site backup and disaster recovery planning</li>
<li>designing, developing and integrating computer code with other specialised inputs, such as image files, audio files and scripting languages, to produce, maintain and support web sites</li>
<li>assisting in analysing, specifying and developing Internet strategies, web-based methodologies and development plans</li>
<h2>Eligible applying visa subclasses</h2>
<p>This occupation belongs to Short-term Skilled Occupation List (STSOL) . STSOL consists of the list of the occupations which are high in demand in Australia. If you want to apply for Australia sponsored Permanent Residency (subclass 190) your occupation must be listed in the relevant STSOL list and you must have an invitation/sponsorship from the state government. This list is also a part of the Australia Point system. If eligible, you can live and work in Australia for an indefinite period of time.</p>
<p>If your occupation is on STSOL, you may be eligible to apply for the following visa subclasses:</p>
<li>Skilled Nominated visa (subclass 190)</li>
<li>Training visa (subclass 407)</li>
<li>Skilled Work Regional (Provisional) visa (subclass 491) - State or Territory nominated</li>
<li>Temporary Skill Shortage visa (subclass 482) - Short-term stream</li>
<li>Skilled Employer Sponsored Regional (Provisional) visa (Subclass 494)</li>
